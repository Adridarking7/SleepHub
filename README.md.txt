# 🏨 Aplicación Web – Gestión de Hotel "SleepHub"

Proyecto desarrollado como **Trabajo de Fin de Grado** del ciclo formativo de **Desarrollo de Aplicaciones Web (DAW)**.

## 📋 Descripción

Aplicación web completa para la gestión de un hotel con funcionalidades como:

- Gestión de **reservas**, **habitaciones**, **usuarios** y **reseñas**
- Sistema de autenticación y control de acceso con **roles**:  
  - Administrador  
  - Editor  
  - Cliente
- Panel de administración para control interno del hotel
- Diseño responsive y estructura clara

## ⚙️ Tecnologías utilizadas

- **Java**
- **Spring Boot**
- **Thymeleaf**
- **HTML5 / CSS3 / JavaScript**
- **MySQL**
- **Git**

## 🧠 Funcionalidades destacadas

- CRUD completo (crear, leer, actualizar, eliminar)
- Gestión de roles y permisos
- Interfaz visual limpia y adaptable
- Integración de diseño personalizado para el hotel
- Separación por capas (MVC)
